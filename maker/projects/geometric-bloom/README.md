# Geometric Bloom Generator

Generative flower-like geometric patterns with rotating petals and color cycling. Each bloom is unique with procedural petal counts, colors, and rotation speeds.

## Demo

Open `output/geometric-bloom.html` in a browser.

## Features

- **Procedural Generation**: Each bloom has unique petal rings, colors, and animations
- **6 Color Themes**: Aurora, Sunset, Ocean, Forest, Monochrome, Neon
- **Interactive Controls**:
  - Click canvas or "New Bloom" button to generate new pattern
  - "Theme" button cycles through color schemes
  - "Pause" button or Space bar to pause/resume animation
- **Background**: Twinkling stars that create depth

## Controls

| Input | Action |
|-------|--------|
| Click canvas | Generate new bloom |
| Space | Pause/resume |
| Theme button | Cycle colors |

## Technical Details

- Pure HTML5 Canvas (no dependencies)
- Bezier curves for smooth petals
- HSL color cycling
- Radial gradients for depth
- ~800 lines of code
