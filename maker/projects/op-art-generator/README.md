# OP Art Generator

Optical illusion patterns inspired by Bridget Riley and Victor Vasarely.

## Features
- **Waves**: Sinusoidal wave patterns that create motion
- **Concentric**: Pulsating concentric shapes
- **Radiating Lines**: Starburst patterns with movement
- **Chevron**: V-shaped repeating patterns
- **Breathing Circles**: Expanding/contracting rings
- 6 color themes

## Controls
- **Click**: Regenerate pattern
- **Space**: Pause animation
- **Arrow Up/Down**: Change pattern type

## Run
Open `output/op-art.html` in browser.
